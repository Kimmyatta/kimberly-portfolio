# Kimberly Atta-Peters Portfolio

A professional React + TypeScript portfolio website for Kimberly Atta-Peters, focused on data science, AI engineering, machine learning, MLOps, and analytics roles.

## Project Structure

- `portfolio_source.md` - human-readable source content for the portfolio
- `src/data/portfolio.ts` - structured content used by the React app
- `src/components/` - reusable portfolio sections and cards
- `src/App.tsx` - page composition and navigation
- `src/App.css` - responsive site styling

## Sections

- Hero
- About
- Skills
- Projects
- Experience
- Education
- Resume
- Contact

## Content Updates

Portfolio content is managed in `src/data/portfolio.ts`.

Update contact information, project links, experience bullets, and resume metadata there.
